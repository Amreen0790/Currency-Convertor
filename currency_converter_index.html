Make & Publish an HTML App on GitHub Pages

One-page, step-by-step instructions with a tiny input example and the expected output URL.

A) Input (what you prepare)

Create a file named index.html (lowercase, exact name). Start with this minimal page:

<!doctype html>
<html>
  <head>
    <meta charset="utf-8" />
    <meta name="viewport" content="width=device-width, initial-scale=1" />
    <title>My Web App</title>
  </head>
  <body>
    <h1>Hello, GitHub Pages!</h1>
    <p>If you see this online, your site works üéâ</p>
  </body>
</html>

Why ‚Äúindex.html‚Äù? GitHub Pages uses it as the default page when someone opens your site.

B) Process (what to click)

  1. Create a repository
    Go to github.com ‚Üí click + ‚Üí New repository ‚Üí name it (e.g., my-web-app) ‚Üí choose Public ‚Üí Create repository.
  2. Upload your HTML
    In the repo, click Add file ‚Üí Upload files ‚Üí drag in index.html (in the root, not inside a folder) ‚Üí Commit changes.
  3. Enable GitHub Pages
    Repo Settings ‚Üí left sidebar Pages ‚Üí Source = Deploy from a branch ‚Üí Branch = main, Folder = / (root) ‚Üí Save.
  4. Wait for deploy
    Give it ~30‚Äì90 seconds. A green banner or the Pages panel will show your site link.

C) Output (what you get)

Your public website URL will look like:

https://<your-username>.github.io/<your-repo-name>/

Open that link ‚Äî you should see Hello, GitHub Pages!.

Input: index.html Output: GitHub Pages URL

D) Update later

  • Edit index.html (or add style.css, app.js), then Commit again.
  • Pages redeploys automatically. Refresh your site URL.
<link rel="stylesheet" href="style.css">
<script src="app.js" defer></script>
<img src="logo.png" alt="logo">

E) Common mistakes (fast fixes)

  • Filename must be index.html (not Index.html or index.html.txt).
  • Location: put it in the root of the branch you publish.
  • Branch: Pages must point to the branch that has your file (usually main).
  • Delay: wait a minute and hard refresh (Ctrl/‚åò + Shift + R).
  • Private repos: use Public while learning for simplest setup.
